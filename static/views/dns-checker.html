What is a DNS Record?

The Domain Name System specifies a database of information elements for network resources. The types of information elements are categorized and organized with a list of DNS record types, the resource records (RRs). Each record has a type (name and number), an expiration time (time to live), a class, and type-specific data. Learn more about DNS and Record Types on the official wikipedia page https://en.wikipedia.org/wiki/Domain_Name_System.
